Scriptum — Editor Minimalista v2.0
https://adagenor2025.github.io/Scriptum
Interface

Visual nostálgico do Mac OS System 7 — desktop xadrez, janela com borda dupla, barra de título listrada
Página A4 com margens (2,5 cm superior, 2,0 cm esquerda) e separação automática entre páginas
Numeração de páginas no canto superior direito — ativável/desativável

Menus

Arquivo — Novo, Abrir, Salvar, Salvar como, Salvar como PDF, Sobre, Encerrar
Inserir — Cabeçalho, Rodapé, Contador de laudas
Editar — Desfazer, Refazer, Copiar, Colar, Selecionar tudo, Negrito, Itálico, Sublinhado, Modo escuro, Tela cheia, Numerar páginas
🔍 Lupa — Busca com realce e navegação entre ocorrências (Cmd+F)
Encerrar — botão visível na barra de menus

Formatação

Negrito, Itálico e Sublinhado (Cmd+B, Cmd+I, Cmd+U)
4 fontes: Courier Prime, Times New Roman, Arial, Georgia
6 tamanhos: 11, 12, 14 (padrão), 16, 18, 22
Alinhamento com ícones SVG modernos: esquerda, centro, direita, justificado

Salvamento

Autosave automático a cada 5 segundos no LocalStorage
Salvar como .txt com nome personalizado
Salvar como PDF via diálogo de impressão do browser

Barra de status

Contador de palavras, caracteres e laudas (2.100 caracteres = 1 lauda)

Infraestrutura

Arquivo único index.html — sem dependências externas
Hospedado no GitHub Pages — acessível por qualquer browser sem instalar nada
Funciona em Windows, macOS e Linux
Compatível com Chrome, Edge, Firefox e Safari
